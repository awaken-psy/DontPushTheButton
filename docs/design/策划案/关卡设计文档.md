# 关卡2



在这里，玩家就要迎来整个开始的决策点了，玩家可以先上去观察第二关的第一个谜题，决策其要超载什么能力，然后在回到维修站选择其超载能力，那么在第一个谜题当中，实际上来说会有非常多种解法，具体的解法需要玩家自己去探索最快的一种解法或者是最节省的一种解法，这些都需要去玩家去破解，去选择他们认为最有可能的路线。在这里主要设计了5种解法，当然想象力是无限的，可能有更多，但目前我自己设计的是四种解法，玩家无论选择超载哪个能力都可以过关。在这边，整个的场景还是一个峡谷地貌相关类型，一些没有围着的区域就是悬崖边，然后

先说明一下解法，在这里我自己预计的解法总共有四个，第一个是移动单格箱子到按钮处，然后右边平台下面的柱子会升上来玩家利用跳跃加移动就可以通过了，第二个解法，直接超载跳跃跳到右边平台上也可以直接上去，第三种解法，利用超载推将更重的箱子推到悬崖边缘然后跳过去也可以，第四种解法在左边这些平台上普通跳跃寻找最佳路径跳过去。

![Image](https://internal-api-drive-stream.feishu.cn/space/api/box/stream/download/authcode/?code=OWY5YjJlZjEzYmEyM2Q5ODlmN2VjZjI4NTg4ZjViYmZfMzhhYzE5YTlkODIyNjAwZDE4MzRiMWQ4NzZhMDUxNjJfSUQ6NzY1NDU4Nzk3NDM1NjU4NTY3NF8xNzgyNzU4ODMxOjE3ODI4NDUyMzFfVjM)

上了这个平台之后就会看到这里有一个谜题需要解，那么这里最表面的谜面实际上就是如果玩家上去利用跳跃按了每一个按钮或者推箱子按了每一个按钮后，每一个符文的图案都会闪烁一定的次数，这个次数代表的就是按键的顺序，当玩家顺着这个按键按了之后大门就会打开，当然这个是最常规的解谜方法。这里主要是设计了一个反常识，实际上玩家可以依靠超载能力推动箱子搭上去，同时也可以利用超载能力推一个箱子跳上去平台，然后直接从上面就能过到对面，没必要打开门。

在这里整个的场景我的构想中会是一个小的遗迹建筑的那种感觉，在这里符号方面美术老师可以自由发挥，不用看我图片里的这些符号，我乱搓的。可以给一个方向，或许这个符文在几何美术上也可以给一定的顺序提示，看美术老师愿不愿意干。

# 关卡3

关卡3开头这里，开头就是新能力pull的能力体现，射桥上的可吸附位置，然后将桥给放下来，如果看起来有点怪的话或者实现有问题的话，也可以将桥放在平台上让玩家吸附过来，因为在这个场景中我预设是一个在水上的一个场景，会是一个个的小岛屿以及礁石这样的一个场景。然后在后边也会有一个超载能力能够走近道的这么一个锚点。

然后到了第二部分，这里整体的设计思路其实也是跟第二关是差不多的，会有多种的解法以及可以利用超载的近道，当然也要付出相应的代价就是了，比如说在这里玩家可以利用pull的超载直接吸过去，也可以推一个箱子过来利用超载跳跳过去。常规过则是需要将柱子障碍物给吸过来然后在通过，并且在过去之后会发现箱子也卡住了，需要返回到入口处把门处的箱子先吸过来然后在通过。

然后第三部分这里就是一个吸附与跳跃能力的一个简单考验，玩家需要通过这些平台以及悬浮的可吸附 平台来搭出属于他们的路。

![Image](https://internal-api-drive-stream.feishu.cn/space/api/box/stream/download/authcode/?code=M2JlZjM3NjliYTM5N2FkMTUzYWFhNTc4ZGNkY2JmOWJfZmQwNWY5NWUzMzk0MTc0YWJiY2ZiZjRhYWU2MDUyMzBfSUQ6NzY1Njg1OTAxNzM2MDM0NjA0MF8xNzgyNzU4ODMxOjE3ODI4NDUyMzFfVjM)

最后一个部分则是一个机关谜题，在这里也是可以又非常多种解法，预留了很多的空间给玩家来释放想象力。

第一种解法：利用拉动能力先将卡在缝里的cube拉出，然后在推cube上两边的斜坡按下按钮然后前面的门就会开始滑动（左边对于第一个滑动的门，右边对应第二个滑动的门，然后这边的这个门是一个弯曲的那种滑轨带点机械感的门，不用这么直，我懒得做了，文字描述一下），玩家需要将两个门的门空隙进行对齐，才能够通过这个点。

第二种解法：玩家如果超载的能力是超载跳，那么可以直接跳上缝的cube，然后在利用pull能力吸一个cube过来然后在跳上去，也可以直接通过

第三种解法：当按下任意一个按钮的时候，门上面的浮块也会进行移动，然后带有吸附锚点的那个就是一个超载捷径。

通过后便来到第四关
