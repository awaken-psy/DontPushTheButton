## API

|功能|API|脚本|未使用|
|---|---|---|---|
|主角在局部Z方向上的速度|Speed|ProtagonistRig||
|更新上列速度|UpdateSpeed\(\)|ProtagonistRig||
|主角接触面法向|Normal|ProtagonistHitInfo||
|天线惯性摇摆|OnLateUpdate\(float, float, float, float, float\)|ProtagonistBody||
|根据接触面法线倾斜底盘|UnderpanSlant\(Vector3\)|ProtagonistUnderpan||
|底盘惯性摇摆（会导致上下坡时玩家模型旋转错误，原因未知）|InertanceSlant\(bool, float, float, float, float, float\)|ProtagonistUnderpan|√|
|旋转左侧轮胎（传入正数向前旋转）|RollLeft\(float\)|ProtagonistUnderpan||
|旋转右侧轮胎（传入正数向前旋转）|RollRight\(float\)|ProtagonistUnderpan||
|激活悬挂（与当前底盘适应斜坡方法配合差）|ActiveSuspend\(bool\)|ProtagonistUnderpan|√|
|旋转螺旋桨|Roll\(float\)|ProtagonistAirscrew||
|旋转引擎风扇|Roll\(float\)|ProtagonistEngine||
|激活重力枪|SetActive\(bool\)|ProtagonistGravityGun||
