## 3.1 推动完整系统（磁力枪·抓取搬运）

> Play mode，键盘。MCP 已验证核心状态机（抓取进 _carried / 吸附前方 carryDistance=1.5 / 释放清空）。配置：Push 绑 E（普通）或超载键。

- [ ] **3.1a 抓取与跟随**
  - 操作：走到 TestCrate 旁（< grabRange=2），按 E
  - 期望：箱子被吸附到玩家前方 carryDistance 处，随玩家移动/转向跟随
- [ ] **3.1b 释放**
  - 操作：持有状态下再按 E
  - 期望：箱子放下（停在当前位置），不再跟随
- [ ] **3.1c 抓取切换稳定**
  - 操作：释放后再按 E（面对箱子），反复切换
  - 期望：Idle↔Carrying 切换稳定，无卡死/重抓异常
- [ ] **3.1d 超载搬多个**
  - 操作：Push 绑超载键，多个 Pushable 在 grabRange 内，按超载键
  - 期望：吸附范围内全部（上限 maxCarryCount=3），横向排列跟随；腐败 +IncrementPerPress 一次
- [ ] **3.1e 边界（抓取失败不扣腐败）**
  - 操作：grabRange 外按 E / 无 Pushable 时按超载键
  - 期望：不抓取（保持 Idle）；超载键也不扣腐败（抓取失败不 ChargeOverload）

## 3.2 拉动能力（磁力枪·牵引 + 超载抓钩）

> Play mode，键盘。MCP 已验证（普通拉吸到前方 pullToDistance=1.5 / 超载抓钩玩家位移到命中点前方 hookLandDistance）。配置：Pull 绑普通键（拉）或超载键（抓钩）。

- [ ] **3.2a 普通拉**
  - 操作：面对 TestCrate（< grabRange=2），按拉动键（普通键）
  - 期望：箱子吸到玩家前方 pullToDistance 处
- [ ] **3.2b 超载抓钩**
  - 操作：Pull 绑超载键，远处（< hookRange=8）面对 Pushable，按超载键
  - 期望：玩家瞬时位移到目标前方 hookLandDistance；腐败 +IncrementPerPress 一次
- [ ] **3.2c 边界（无目标不扣腐败）**
  - 操作：grabRange 外按普通拉 / 无 Pushable 时按超载抓钩
  - 期望：普通拉不吸（无目标）；抓钩不触发也不扣腐败（射线未命中 Pushable）
- [ ] **3.2d 纵深补偿**（俯视 2.5D）
  - 操作：抓钩位移到不同纵深目标
  - 期望：落点安全（不掉沟）；纵深靠阴影/边缘标记补偿（v1.0「调试可接受」）
