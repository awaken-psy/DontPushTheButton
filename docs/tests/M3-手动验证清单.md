## 3.1 拾取能力（磁力枪·抓取搬运 / Pickup）

> Play mode，键盘。MCP 已验证核心状态机（抓取进 _carried / 吸附前方 carryDistance=1.5 / 释放清空）。配置：Pickup 绑 E（普通）或超载键。

- [x] **3.1a 抓取与跟随**
  - 操作：走到 TestCrate 旁（< grabRange=2），按 E
  - 期望：箱子被吸附到玩家前方 carryDistance 处，随玩家移动/转向跟随
- [x] **3.1b 释放**
  - 操作：持有状态下再按 E
  - 期望：箱子放下（停在当前位置），不再跟随
- [x] **3.1c 抓取切换稳定**
  - 操作：释放后再按 E（面对箱子），反复切换
  - 期望：Idle↔Carrying 切换稳定，无卡死/重抓异常
- [x] **3.1d 超载搬多个**
  - 操作：Pickup 绑超载键，多个 Pushable 在 grabRange 内，按超载键
  - 期望：吸附范围内全部（上限 maxCarryCount=3），横向排列跟随；腐败 +IncrementPerPress 一次
- [x] **3.1e 边界（抓取失败不扣腐败）**
  - 操作：grabRange 外按 E / 无 Pushable 时按超载键
  - 期望：不抓取（保持 Idle）；超载键也不扣腐败（抓取失败不 ChargeOverload）

## 3.2 拉动能力（磁力枪·牵引 + 超载抓钩）

> Play mode，键盘。MCP 已验证（普通拉吸到前方 pullToDistance=1.5 / 超载抓钩玩家位移到命中点前方 hookLandDistance）。配置：Pull 绑普通键（拉）或超载键（抓钩）。

- [x] **3.2a 普通拉**
  - 操作：面对 TestCrate（< grabRange=2），按拉动键（普通键）
  - 期望：箱子吸到玩家前方 pullToDistance 处
- [x] **3.2b 超载抓钩**
  - 操作：Pull 绑超载键，远处（< hookRange=8）面对 Pushable，按超载键
  - 期望：玩家瞬时位移到目标前方 hookLandDistance；腐败 +IncrementPerPress 一次
- [x] **3.2c 边界（无目标不扣腐败）**
  - 操作：grabRange 外按普通拉 / 无 Pushable 时按超载抓钩
  - 期望：普通拉不吸（无目标）；抓钩不触发也不扣腐败（射线未命中 Pushable）
- [x] **3.2d 纵深补偿**（俯视 2.5D）
  - 操作：抓钩位移到不同纵深目标
  - 期望：落点安全（不掉沟）；纵深靠阴影/边缘标记补偿（v1.0「调试可接受」）

## 3.4 冲刺能力（dash + 超载 dash×2）

> Play mode，键盘。MCP 已验证（普通 dash=dashDistance=5 / 超载 dash×2=10）。配置：Dash 绑普通键或超载键。

- [x] **3.4a 普通 dash**
  - 操作：按移动方向（如 W 或 D）+ dash 键
  - 期望：朝角色正前方瞬时突进 dashDistance（不管按 W/D/A/S，只要在移动都朝面朝方向冲）；无移动输入不 dash
- [x] **3.4b 超载 dash×2**
  - 操作：Dash 绑超载键，按超载键
  - 期望：突进距离×2；腐败 +IncrementPerPress 一次（J4：1 次触发算 1 次）
- [x] **3.4c 防穿墙**
  - 操作：面对墙 dash
  - 期望：停在墙前 0.1m（不穿墙）
- [x] **3.4d 冷却**
  - 操作：连按 dash
  - 期望：冷却内不触发（cooldown=0.5s）

> Play mode，键盘。MCP 已验证（射线推 TestCrate beamPushDistance=3）。配置：关卡 offer Push（PushAbility，光束型）时，Push 绑键。

- [ ] **3.3a 普通光束推**
  - 操作：面对 Pushable（< beamRange=8），按 Push 键
  - 期望：物体被推 beamPushDistance（朝玩家朝向）
- [ ] **3.3b 超载光束推**
  - 操作：Push 绑超载键，按超载键
  - 期望：推距×2；腐败 +IncrementPerPress 一次
- [ ] **3.3c 二选一（关卡固定）**
  - 操作：关卡 LevelConfigSO offer Pickup（拾取搬运）或 Push（光束推）
  - 期望：该关玩家只能用对应 variant（配置 UI 只显 offer 的）

## 3.5 跳跃超载飞行状态机（+ 缓落）

> Play mode，键盘。MCP 已验证（状态机 IsSlowFalling + ApplyGravity 缓落 FallGravityScale=0.5）。配置：Jump 绑超载键触发飞行。

- [x] **3.5a 超载飞行**
  - 操作：Jump 绑超载键，按超载键起跳
  - 期望：初速×倍率 + 飞行期持续上升（OverloadFlyDuration）；腐败 +IncrementPerPress 一次
- [x] **3.5b 缓落**
  - 操作：超载飞行结束后观察下落
  - 期望：下落速度减半（FallGravityScale=0.5，缓落感）；落地恢复正常
- [x] **3.5c 普通跳**
  - 操作：Jump 绑普通键，按起跳
  - 期望：√(2gh) 正常跳，正常重力落（不缓落）
- [x] **3.5d 飞行时长可调**
  - 操作：改 JumpTuning._overloadFlyDuration，playtest
  - 期望：飞行时长随之变（SO 调参）
