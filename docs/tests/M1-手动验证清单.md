## 1. 角色控制（M1.4）

> 进 Play mode（▶ 或 Edit → Play），用键盘操作。俯视 2.5D 视角。

### 1.1 移动
- [x] **WASD 移动响应**
  - 操作：依次按 W / A / S / D
  - 期望：角色朝四方向移动；**方向相对相机水平面**（W = 画面向远处/上方，与相机朝向一致）
- [x] **移动速度手感**
  - 操作：长按 W 观察快慢
  - 期望：moveSpeed=5 手感适中（不拖沓、不失控）。过快/过慢 → 记下，改 `MovementTuning._moveSpeed`
- [x] **斜向移动**
  - 操作：同时按 W + D（等任意两键组合）
  - 期望：角色朝斜向移动，速度与单向一致（输入已 normalize，不会斜向更快）
- [x] **转向**
  - 操作：快速改变移动方向（如 W→D→S）
  - 期望：角色平滑转向移动方向（turnSpeed=720），不是瞬切；停止移动时保持最后朝向

### 1.2 跳跃与重力
- [x] **起跳**
  - 操作：按 Space
  - 期望：角色起跳；跳跃高度 ≈ jumpHeight=1.5m（约大半个胶囊高）
- [x] **重力下落**
  - 操作：起跳后松开所有键
  - 期望：角色自然下落回到地面，**无漂浮感、无瞬落**
- [x] **落地稳定不抖**
  - 操作：落地后静止站立数秒
  - 期望：角色贴地稳定，**不上下抖动**（贴地速度钳制 isGrounded 生效）
- [x] **空中不二段跳**（当前设计）
  - 操作：起跳到最高点前再按 Space
  - 期望：**不触发二段跳**（当前仅在 grounded 时起跳）。若后续要改 Coyote 时间/空中跳，记下

### 1.3 俯视 2.5D 相机
- [x] **相机跟随位置**
  - 操作：移动角色四处跑
  - 期望：相机始终跟随角色位置（角色大致在画面中心）
- [x] **相机角度固定**
  - 操作：移动 + 转向角色
  - 期望：相机俯角 ~50° **固定不变**，**不随角色转向旋转**（Aim 固定）
- [x] **相机不穿地板/不抖**
  - 操作：角色跑到场景边缘、来回快速移动
  - 期望：画面稳定，相机不陷入地板、不抽搐（无阻尼 = 硬跟随，应贴住角色）

### 1.4 落点标记（俯视纵深补偿 / GDD 锁定需求）
- [x] **静止时圆盘在脚下**
  - 操作：角色站立不动
  - 期望：黄色扁平圆盘在角色正下方地面，位置对齐
- [x] **跳跃时圆盘留在落点**（核心）
  - 操作：原地起跳
  - 期望：角色浮起，**黄色圆盘留在原地面位置不动**；俯视下能清晰看出「角色会落回这里」
- [x] **移动时圆盘实时跟随**
  - 操作：移动角色
  - 期望：圆盘实时贴在角色 XZ 投影下方（LateUpdate 同步）

## 附录 A：调参速查（测试中发现手感不对时）

| 现象 | 调整项 | 位置 |
|------|--------|------|
| 移动太快/慢 | `_moveSpeed` | `Assets/Config/MovementTuning.asset` |
| 跳太低/高 | `_jumpHeight` | 同上 |
| 下落太慢/快 | `_gravity` | 同上 |
| 转向太慢/瞬移 | `_turnSpeed`（0=瞬切） | 同上 |
| 相机太远/近 | `Distance`（builder 常量 14）或 CM TopDown 的 `m_CameraDistance` | `CinemachineTopDownBuilder.cs` / `CM TopDown` |
| 相机俯角不适 | `Pitch`（builder 常量 50）或 CM TopDown Transform 旋转 X | 同上 |
| 角色卡台阶 | `stepOffset` / `_jumpHeight` | CharacterController / MovementTuning |

> 改 MovementTuning 数值无需改代码，Play 中改 `.asset` 即时生效（重启 Play 更稳）。改相机 Pitch/Distance 需改 builder 常量后重跑菜单 `DPTB/Build Top-Down Camera`，或直接调场景里 `CM TopDown` 的 Transform 旋转 / FramingTransposer.m_CameraDistance。

## 附录 B：失败排查

| 现象 | 排查 |
|------|------|
| 角色完全不动 | ① Console 有无 `[PlayerControls] InputActionAsset 未赋值`（检查 `_actions` 已赋 `PlayerControls.inputactions`）；② Active Input Handling 是否 = New（ProjectSettings，1.3 设过，重启生效）；③ PlayerMover.`_tuning` 是否赋值 |
| 角色穿地 / 抖动 | ① CharacterController 的 height/radius/center 是否罩住胶囊；② Floor 有 BoxCollider；③ stepOffset 是否过大 |
| 相机不动 / 角度错 | ① Main Camera 是否挂 `CinemachineBrain`；② `CM TopDown.Follow` 是否 = Player；③ Pitch/Distance 常量；④ 是否多个 vcam 抢占 |
| 材质粉色（丢失） | shader 不是 URP——应为 `Universal Render Pipeline/Lit`（落点圆盘是黄色 Lit） |
| 圆盘不跟随 / 漂浮 | `LandingMarker._target` 是否 = Player；`_groundY` 是否 = 地面高度 |
