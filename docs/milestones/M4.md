## 任务详述

### 4.4 全 5 关 playtest 调参（最高优先）
- **做什么**：对**全 5 关** playtest——关 1（onboarding 是否清晰）、关 2/3/4（双解法两条路线都有人走）、关 5（终局节奏）。若双解法都走超载 → 调高常规繁琐度 / 腐败代价；都不走 → 反向调。
- **依据**：`toConfirm §J8`；GDD `7`；**S3（扩全 5 关，含关 1/5）**
- **为什么最优先**：验证 M2–M3 堆出来的核心乐趣是否真的成立
- **验收**：全 5 关 playtest 记录；双解法关两条路线都有人走
- **依赖**：M3.6

### 4.1 腐败层级机制微调（playtest 后）
- **做什么**：原「机关反转 + 重力波动」**v1.0 已移除**（改腐败 5 层级，见 M3.8）。本任务改为：playtest 后根据反馈微调腐败层级的惩罚幅度（移速降 / glitch / 冷却延长数值）。
- **依据**：GDD `4.4` 腐败层级；v1.0 整合（撤销 §J6）
- **验收**：腐败层级惩罚手感经 playtest 调整到位
- **依赖**：M3.8 + 4.4 playtest

### 4.2 配置阶段 3D 实时预览
- **做什么**：配置阶段的关卡预览从「无/静态截图」升级为 **3D 实时预览 + 相机轨道控制**，玩家可旋转查看障碍布局。
- **依据**：GDD `4.1`（关卡全貌预览）；`toConfirm §J6`
- **验收**：配置时能 3D 旋转查看关卡
- **依赖**：M2.4

### 4.3 关卡选择界面（P2）
- **做什么**：非线性关卡选择菜单（替代纯线性推进），显示每关星级 / 最佳用时 / 解锁状态，支持重玩刷星。
- **依据**：GDD `4.8`；`toConfirm §E`（P2 后置）
- **验收**：关卡选择菜单可用、星级持久
- **依赖**：M3.7

### 4.5a 数值精调
- **做什么**：全局数值精调（腐败增量 / 强化倍率 / 跳跃距离 / 飞行时长 / 节制阈值 / dash 语义）。
- **依据**：GDD `7`；`toConfirm §H`
- **验收**：数值手感到位
- **依赖**：M3.6 / M3.7

### 4.5b 结局（解谜化 + 动画）
- **做什么**：关 5 终点——玩家把「发送信号」能力绑到某键按下触发结局（键位配置主题收尾），然后通讯阵列点亮 + 信号射向星空动画。
- **依据**：GDD `2.5`（结局解谜化）；v1.0 整合（吸收队友）
- **验收**：结局按键解谜 + 动画都实现
- **依赖**：M3.6（关 5 跑通）；可与 4.5a 并行
