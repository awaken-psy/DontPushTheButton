## 任务详述

### 3.0 纸面可解性证明（关 1 / 3 / 4 / 5）—— 最高优先前置
- **做什么**：**纯设计推演，不写代码**。每关产出一份 `docs/design/level_X_solution_proof.md`：障碍清单 × 所需能力 × 至少 2 个可解配置。重点证明：
  - **关 4 双解法可解**：至少一个 `(移动4 + 冲刺超载 + 跳/推/拉中恰 2 个)` 配置可解（`§J1`）
  - **关 5 代表性弃法可解**：验 3–4 种典型弃法（如弃后退 / 弃冲刺 / 弃拉动）可解（`§J1`、**R3 决策**）
- **依据**：`§J1`（关 4/5 可解性）、`§J4`；**R2（开发前证明，非白盒后）、S1（产物定义）、R3（代表性弃法）**
- **为什么前置**：在写关卡代码前消除死局风险，0 代码成本，回报最高（避免白盒返工）
- **验收**：4 份 proof 文档完成，关 4/5 可解性有明确结论

### 3.1 推动完整系统（磁力枪·抓取搬运型）
- **做什么**：把 M2.1 的最简推动升级为完整能力——磁力枪·抓取模式，近距吸附搬运；超载 = 同时搬多个。
- **依据**：GDD `4.2`（推动·搬运型）；`§B`、`§A` 推荐
- **验收**：吸附搬运 + 超载搬多个
- **依赖**：M2.1

### 3.2 拉动能力（磁力枪·牵引）
- **做什么**：拉动——发射磁铁黏目标 → 再按吸向自己；超载 = **反向把自己吸向目标（抓钩 / 自身位移）**。（v1.0 恢复抓钩，撤销 §J3；俯视 2.5D 下调试可接受。**fallback**：进度告急则降级为拉更远/多个，关 3 解法 B 同步改为「拉石柱更远腾路」）
- **依据**：GDD `4.2`；`§B`、`§J3`
- **验收**：拉物体 + 超载拉更远/多个
- **依赖**：M2.1

### 3.3 推动二选一：牵引光束型
- **做什么**：推动第二种形态——牵引光束远距推物；超载 = 推更远更快（数值）。配置阶段与搬运型二选一（用 M2.4 预留的 `ability.variants`）。
- **依据**：GDD `4.2`；`§B`；**R7（依赖 M2.4 接口预留）**
- **验收**：配置时能选光束型 + 光束推物 + 超载数值
- **依赖**：3.1

### 3.4 冲刺能力
- **做什么**：冲刺 = dash；超载 = dash×2。关 4 引入。
- **依据**：GDD `4.2`；`§B`
- **验收**：冲刺位移 + 超载连 dash 两次
- **依赖**：M2.1

### 3.5 跳跃超载飞行状态机（完善）
- **做什么**：把 M2.5 的瞬时短飞升级为**持续飞行状态机 + 缓落 + SO 调参**（时长/手感可调）。
- **依据**：GDD `4.2`；`§B`、`§H`；**H3（与 M2.5 划界：本任务做状态机+缓落+SO，M2.5 只做瞬时写死版）**
- **验收**：飞行时长可调 + 缓落手感可接受
- **依赖**：M2.5

### 3.6 关 1 / 3 / 4 / 5 白盒
- **做什么**：白盒搭建：
  - **关 1（教学·苏醒）**：纯移动走到出口，onboarding（**H1 补位**）
  - 关 3（拉动·信号干扰）
  - 关 4（冲刺·断桥·双解法）
  - 关 5（终局·双超载·必弃 1）
- **依据**：GDD 第 5 章 + 第 6 章；`§D`；**3.0 纸面证明（关 4/5 可解性已验）**
- **关键（防认知偏差）**：
  - 关 4 双解法必须可解（引用 3.0 proof）
  - **关 5「代表性弃法可解」**（3–4 种典型，引用 3.0 proof，**R3 决策**；不再要求「任意弃 1」）
- **验收**：4 关各自可解路径跑通
- **依赖**：3.0 + 3.1–3.4

### 3.7 评分系统（4 星）
- **做什么**：4 星（通关 / 时间 / 节制 / 精简）+ 软计时 + HUD + 结算界面（评分挂载点）。**每关节制阈值独立 SO 字段**（关 4 可独立放宽，**R6**）。
- **依据**：GDD `4.7`；`§F`；`§J4`；**R6**
- **不做**：关卡选择界面（M4.3，P2）
- **验收**：过关结算显示星级；4 星可追
- **依赖**：M2.8

### 3.8 腐败 URP 视效
- **做什么**：腐败 5 层级（GDD 4.4）——视效（泛红 / glitch 抖动 / 屏幕抖动，URP Volume）+ 音效（不和谐干扰）+ **机制层**（层级 2 移速降、层级 3 能力冷却延长）。机制层涉及角色控制器 + 能力系统。
- **依据**：GDD `4.4`；支柱 4
- **不做**：机关反转 / 重力波动（v1.0 已移除，改本任务的层级设计）
- **验收**：腐败层级有清晰视觉/听觉反馈
- **依赖**：M2.6

### 3.9 场景主题（至少 3 个）
- **做什么**：至少 3 个可复用户外场景主题（坠毁点 / 荒原 / 古代遗迹 / 母舰残骸中选 3+，具体待美术）。**全室外**。
- **依据**：GDD `2.3`；`§D`；**S5（与 GDD §9「4 主题」统一为「至少 3 个」，GDD 同步降措辞）**
- **验收**：5 关有差异化场景外观
- **依赖**：3.6
